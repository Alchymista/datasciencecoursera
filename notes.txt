Tyepes of Data Science Questions
================================
Descriptive - goal is to describe a set of data (usualy after some statistical modeling)
Exploratory - find relations you didn't know about
Inferential - small sample of data to say soething about a bigger population
Predictive - use data on some objects to predict values for another
Casual - what happens to one variable when you make another variable change
Mechanistic - understand the exact changes in variable that lead to changes in other variables for individual objects
